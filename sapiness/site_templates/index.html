What is this

Sapiness is a SAP emulator built using Django.

Abippity is an ABAP interpreter written in Python.

SAP (Systems Applications and Products in Data Processing) is an ERP.

ABAP (Advanced Business Application Programming) is SAP's built-in proprietary programming language.

What do I do with it

Go look at the example programs and run them.

Go look at other users' public programs and run them.

Try making a copy of a program and making changes to it. (There's a [Copy] link at the top of every document.)

Create your own program, find an ABAP tutorial, and see how many of its code examples you can copy-paste into this thing and successfully run.

Sign in or sign up for a user account, so you can make your documents (programs) private. (Anonymous users' documents can be changed by anybody.)
